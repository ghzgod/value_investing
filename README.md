# The Value Investor

A static, client-side tool for learning value investing with real market data.
Look up any US-listed stock's Benjamin Graham intrinsic value against its live
price, see the formula worked out with the company's real numbers, play with the
inputs, and backtest the value-timing strategy. No backend – prices come from
Yahoo Finance via CORS relays, the AAA bond yield from FRED, and S&P 500
fundamentals from a bundled snapshot.

Educational only. Not investment advice.
